## How to run

### Install the following packages:

downloader, utils

### Copy and execute the project:

-   Clone the project to a local repository via:

``` r
system("git clone https://github.com/DanielBonnery/ensae-macro.git")
```

-   open the .rproj file in Rstudio
-   run:

``` r
source(demo/main.R)
```
